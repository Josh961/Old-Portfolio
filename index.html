MIPS CPU

Created in Logisim, this MIPS CPU was one of the coolest and most extensive projects I've done at Pitt. It supports 26 MIPS assembly instructions. Pictured is the main CPU itself, the Arithmetic Logic Unit (ALU), and the control unit which reads in a specific Opcode, then executes the appropriate instruction.

GitHub: github.com/Josh961/CS-0447
